# Gmail下書き自動作成ツール（GAS×スプレッドシート）

Google Apps Scriptを使って、スプレッドシート上の連絡先とテンプレートから  
Gmailの下書きを自動で生成するツールです。

---

## 📁 構成ファイル

| ファイル名                 | 内容                             |
|---------------------------|----------------------------------|
| メール自動化スクリプト.gs     | Gmailの下書きを自動生成するメインスクリプト |
| 名刺データ（サンプル）.csv    | 連絡先データ（サンプル）               |
| テンプレ一覧.csv            | 件名・本文のテンプレート一覧            |

---

## 🛠 動作手順

1. **Googleスプレッドシートを作成**

2. **名刺データ（サンプル）.csv をインポートして、シート名を「data」にする**

3. **テンプレ一覧.csv をインポートして、シート名を「config」にする**

4. **スプレッドシートのメニューから  
   「拡張機能 → Apps Script」を開く**

5. **メール自動化スクリプト.gs の内容を貼り付けて保存する**

6. スプレッドシートに戻ると、上のメニューに **「自動化」** が追加されているので、  
   **「下書き作成」** をクリック

7. Gmail の「下書き」フォルダに生成されたメールが入っていれば成功！

---

## 🧠 学べること
- GASでスプレッドシートを自動処理する方法  
- GmailAppで下書きを生成する方法  
- 条件分岐（if文）によるテンプレ切替

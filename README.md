# Gmail下書き自動作成ツール（GAS×スプレッドシート）

Google Apps Scriptを使って、スプレッドシート上の連絡先とテンプレートから  
Gmailの下書きを自動で生成するツールです。

---

## 📁 構成ファイル

| ファイル名                 | 内容                             |
|---------------------------|----------------------------------|
| メール自動化スクリプト.gs     | Gmailの下書きを自動生成するメインスクリプト |
| 名刺データ（サンプル）.csv    | 連絡先データ（サンプル）               |
| テンプレ一覧.csv            | 件名・本文のテンプレート一覧            |

---

## 🚀 動作手順
1. Googleスプレッドシートを作成  
2. data/configシートをそれぞれインポート  
3. Apps Scriptを開いてgas_create_drafts.gsを貼り付け  
4. 「自動化 → 下書き作成」をクリック  

---

## 🧠 学べること
- GASでスプレッドシートを自動処理する方法  
- GmailAppで下書きを生成する方法  
- 条件分岐（if文）によるテンプレ切替
